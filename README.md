
🚚 Courier Management System

A full-stack courier and logistics management platform built with React, Tailwind CSS, Node.js, Express.js, and MongoDB.

The Courier Management System is designed to streamline shipment management for customers, delivery agents, and administrators. It provides a centralized platform for creating shipments, tracking parcels, managing deliveries, assigning agents, and monitoring courier operations in real time.

📖 Overview

Modern courier companies require efficient systems to manage package delivery lifecycles, tracking updates, and operational workflows. This project aims to replicate the core functionalities of a real-world logistics platform while following scalable software architecture and industry-standard development practices.

The application supports multiple user roles:

Customers can create and track shipments.
Delivery Agents can manage assigned deliveries and update shipment statuses.
Administrators can oversee the entire courier operation, manage users, and monitor deliveries.
